runtime总结

發表於 2016-03-09

一、runtime简介

  • Runtime 简称运行时。OC就是运行时机制,其中主要就是消息机制。
  • 对于C语言，函数在编译期就会决定调用哪个函数。
  • 对于OC函数，属于动态调用，在编译期并不能真正决定调用哪个函数，只有在真正运行时候才会根据函数的名称找到对应的函数来调用。
  • 事实证明
    • 在编译阶段，OC可以调用任何函数，即使这个函数并未实现，只要声明过就不会报错。
    • 在编译阶段，C语言调用未实现的函数就会报错。
